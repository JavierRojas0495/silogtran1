🔗 Navegación entre Páginas

Flujo de navegación:

  1. Index → Login (automático)
  2. Login → Forgot Password
  3. Forgot Password → Login (botón "Volver al Login")
  4. Two Factor Auth → Login (botón "Volver al Login")
  5. Dashboard → Manifest
  6. Dashboard → Remesa
  7. Manifest → Dashboard
  8. Remesa → Dashboard
  9. Remesa → Manifest

🔐 Flujo de Autenticación

Para probar el flujo completo:

  1. Ve a Login: vistas/login/login.html
  2. Ingresa credenciales: admin / admin
  3. Te redirigirá automáticamente a Two Factor Auth
  4. Ingresa cualquier código de 6 dígitos (ej: 123456)
  5. Te redirigirá a Cost Center
  6. Selecciona un centro de costos
  7. Te redirigirá al Dashboard

Nota: Este es un flujo simulado para demostración.
